NTFS (New Technology File System) - это файловая система, которая используется в Windows. Она предоставляет полный набор функций, включая дескрипторы безопасности, шифрование, квоты дисков и расширенные метаданные ¹. 

SMB (Server Message Block) - это протокол, который используется для обмена файлами и принтерами между компьютерами в сети. Он позволяет пользователям совместно использовать файлы и папки на удаленном сервере ².

Файловый сервер NTFS/SMB для внутреннего использования - это сервер, который использует файловую систему NTFS и протокол SMB для обмена файлами и папками внутри организации. Он может быть настроен для обеспечения повышенной надежности, безопасности и производительности ¹³⁴.

Например, файловая система NTFS обеспечивает повышенную надежность благодаря использованию файла журнала и сведений о контрольных точках для восстановления согласованности файловой системы при перезагрузке компьютера после сбоя системы ¹. Она также отслеживает и исправляет временные проблемы повреждения в фоновом режиме без отключения тома ¹. 

Протокол SMB обеспечивает безопасность контроля доступа и поддержку шифрования дисков BitLocker ². Для обеспечения безопасности контроля доступа NTFS позволяет задать разрешения для файла или папки, указать группы и пользователей, доступ к которым требуется ограничить или разрешить, и выбрать тип доступа ¹. Шифрование дисков BitLocker обеспечивает большую безопасность критически важных сведений о системе и других данных, хранящихся в томах NTFS ¹.

Для настройки производительности файлового сервера SMB можно изменить размеры буферов и кэшей, используемых протоколом SMB ³. При использовании файлового сервера SMB 3.0, который появился в Windows Server 2012, можно воспользоваться новыми функциями, такими как поддержка множества потоков и улучшенная производительность ⁴⁵.
